# Voice Transcribe Skill（语音转文字 + 语境纠错）

当收到语音消息时，自动完成：下载音频 → Whisper 转录 → **结合历史语境纠错** → 直接回复。

## 工作流程

1. **检测语音消息**：从飞书消息事件中提取 `audio` 类型消息的 `message_id`
2. **下载音频**：使用 `feishu_im_bot_image` 工具下载音频文件（格式：ogg/opus）
3. **Whisper 转录**：调用 `faster-whisper` 本地模型（base）转成文字
4. **获取历史消息**：通过 `feishu_im_user_get_messages` 获取最近 N 条上下文消息（用于语境纠错）
5. **AI 语境纠错**：对转录原文结合历史对话进行纠错
6. **直接回复**：基于纠错后文字直接回复用户，**不输出纠错过程**

## 纠错 Prompt

```
你是一个中文语音转文字的纠错专家。以下是一段语音转录的原始文字，可能存在错别字、标点错误、口误等问题。

结合以下历史对话上下文，对这段语音转录进行纠错。只输出纠错后的文字，不要解释，不要备注，直接输出修正后的完整文字。

历史上下文：
{从飞书获取的最近3-5条消息，格式：发送者: 内容}

原文：{Whisper转录的原始文字}
纠错后：
```

## 回复策略

**如果转录基本可读、语义完整：**
→ 直接基于纠错后文字回复，不提及转录/纠错过程

**如果转录有明显错误、语义缺失、不知所云：**
→ 回复「我好像没听清楚，你说的是……吗？」或「你说的哪部分我没听明白？」，不要胡乱猜测

**如果完全无法识别：**
→ 回复「声音不太清楚，我没听懂你说的，能再发一遍或者打字吗？」

## 转录脚本

```bash
python3 <Skill Location>/scripts/whisper_transcribe.py <音频文件路径> [输出txt路径]
```

### 输出格式

```
语言: zh，时长: Xs
----------------------------------------
[0.0s -> 3.1s] 转录片段1
[3.1s -> 6.2s] 转录片段2
----------------------------------------
保存到: <输出路径>
最终文字: <合并的完整文字>
```

## 获取历史消息

使用 `feishu_im_user_get_messages` 获取私聊历史：

```python
feishu_im_user_get_messages(
    open_id="ou_xxx",  # 发送者 open_id
    page_size=5,        # 获取最近5条
    sort_rule="create_time_desc"
)
```

返回的消息中过滤出 `text` 类型作为上下文。

## 依赖

```bash
pip3 install faster-whisper
```
